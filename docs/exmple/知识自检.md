# 记忆

## **识别、回忆、列举、描述**：

定义、概念、术语、事实、逻辑链等等，用自己的话描述可也助于理解

**方法**：复述、复习笔记、闪卡、大声背诵朗读、记忆术\法

**检验**：复述、选择题、匹配题、填空题

**评价**：人脑数据库基础，金字塔的最底端

# 理解

## **解释、概述、转述**：

用自己方式说明解释和描述记忆的内容、核心概念
多角度视角，多层面描述解释，不同方向的类比[^例子][^费曼技巧]

**方法**：
做摘要、举例书面、类比对比相似概念、向他人讲解

















































[^例子]:比如数学中的物理方法，又或代数和几何两个对同一概念的视角等等
[^费曼技巧]:[费曼技巧](https://methodologywiki.com/zh/learning_methods/Feynman-Technique-Tutorial-zh/)::对完全没有相关背景知识的12岁孩子(第一次来到地球的火星人)解释这个概念，**用自己的、最简单的、最清晰的语言**，写下对概念的理解
