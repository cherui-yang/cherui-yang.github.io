# 问题排序（从基础到综合，从理论到实践）

1. 概念/定义/发明/模块 etc，A是什么？
2. 为什么A是这样的？A的原理/结构是什么？
3. 怎么推导/发展出来的？
4. 推导/发展方式只有这一种吗？
5. 核心是什么？
6. 核心创新点是什么？
7. 和过去不同的是什么？
8. 若把状况推进到任意参数或者无穷会怎么样？
9. 那5中的状况（状态）下的规律又是什么呢？
10. 当A受到噪声、扰动或信息不完整时，它的表现如何？是否具有鲁棒性？
11. A和其它知识/体系/社会/政治/经济之间又有什么联系/关联？
12. A隐含了哪些价值判断或意识形态前提？A的使用应该受到什么样的伦理约束？
13. A对现实有什么反作用？
14. A在不同体系/领域中各有什么特点/相似/不同/应用？
15. 它是其它知识一些总和？
16. 还是组合推导出的新概念或量度有新的意义？还是完全新的A？
17. 为什么会出现/发展出A，它的历史唯物主义路径是怎样的？
18. A有什么用？A用在哪？A能干嘛？A表示/意味什么？A怎么应用？
19. 如何度量/评价A的好坏？有哪些关键指标？这些指标本身有什么缺陷？
20. 有哪些适合搭配A的C 理论/工具/etc？
21. A有什么局限？A经过发展有什么变体？
22. 在什么条件下A会失效或完全不适用？有没有经典的反例？
23. 目前A有什么局限？未来A是怎么样？
24. 目前出现过有替代或者比A更好的B吗？未来呢？
25. A相比B又各有什么优劣？
26. 你能利用A干什么？A对你来说是黑箱吗？
27. 你能自己出/说一些用到A解决的问题/项目吗？或相关竞赛/考试/书后题等等自我检验的库吗？
28. 根据你自身的体验对A有什么想法？（改进/个性化）
29. 初学者（或外行）对A最容易产生的误解是什么？为什么会产生这些误解？
30. 有没有一个直观的图景、比喻或思维实验能帮助理解A的本质？
31. 假如按照费曼学习法，对一个完全没有相关知识的10岁孩子，如何循序渐进，阶梯式的学习、记忆理解应用分析评价创造A，然后精通？它的路径/指南是什么？（但是本人是大年龄，只是用费曼方法）
